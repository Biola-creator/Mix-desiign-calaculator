# Mix-desiign-calaculator
I'm tryin to create a calculator that concrete mix design calculator 
Mix Design Calculator MVP

Run locally with:
streamlit run mix_design_mvp.py

What it does:
- Lets the user choose SI or US units
- Lets the user choose mortar or concrete
- Estimates material quantities based on workability, w/b, binder blend, and batch volume
- Outputs batch quantities, rate per selected unit, total cost, embodied carbon, and warnings
- Allows editing specific gravity, cost, and carbon assumptions inside the app

Important:
- This is an MVP using editable starter assumptions
- Concrete logic is ACI-inspired, not a full code-compliant mix-design package
- Final mix acceptance should come from trial batching and testing
